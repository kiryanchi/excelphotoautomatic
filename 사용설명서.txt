※ 프로그램 설명 ※

    - 공사 작업 전, 후, 전주 번호를 쉽게 넣을 수 있는 프로그램 v 0.4


※ 사용법 ※

    - 첫 작업인 경우

        1. 엑셀 파일 불러오기 (Ctrl + O) 를 눌러 작업하고 싶은 엑셀 파일을 선택한다.

        2. 사진을 넣고싶은 칸을 클릭한 후, 사진을 표로 끌어넣는다.

        3. 버튼 설명을 보고 알맞게 작업 한 후, 엑셀에 집어넣는다.


※ 단축키 ※

    1. Ctrl + O : 엑셀 파일 불러오기

    2. Ctrl + S : 작업 저장

    3. Ctrl + L : 작업 불러오기

    4. Ctrl + C : 사진 복사

    5. Ctrl + V : 사진 붙여넣기

    6. Ctrl + R : 사진 새로고침

    7. Delete : 사진 삭제


※ 버튼 설명 ※

    1. 엑셀 파일 불러오기 (Ctrl + O)

        - 작업하고 싶은 엑셀 파일을 불러온다.


    2. 작업 저장 (Ctrl + S)

        - 작업중인 상황을 xpa, xpae 파일로 저장한다.

        주의) 사진이 들어있는 경로를 바꾸지 말 것 -> 추후 수정 예정


    3. 작업 불러오기 (Ctrl + L)

        - 저장된 xpa, xpaed 파일을 불러와 작업을 진행한다.

        주의) 사진이 들어있는 경로를 바꾸지 말 것 -> 추후 수정 예정

    4. 사진 삭제 (Delete)

        - 선택한 사진을 지운다.

    5. 사진 전체 삭제

        - 표 안의 사진을 모두 지운다.
    
    6. 사진 새로고침 (Ctrl + R)

        - 창의 크기를 바꿨다면, 이 버튼을 눌러 사진 크기를 창에 맞게 조정할 수 있음

    6. 엑셀에 사진 넣기

        - 사진 배치가 다 끝난 후, 엑셀에 사진을 삽입한다.


※ 주의 사항 ※

    1. 사진 경로를 바꾸지 말 것( 바꿨다면 다시 작업해야함 ) -> 수정 예정

    2. 작업 중인 엑셀 파일은 main.exe 와 같은 경로에 있어야 함

        ex) 작업 중인 엑셀 파일이 my_excel.xlsx 라면,
        MyFolder
            ㄴ UI
                ㄴ my_form.UI
            ㄴ main.exe
            ㄴ my_excel.xlsx

※ 수정 및 구현 예정 ※

    1. 사진 경로에 상관없이 작업할 수 있도록 수정

    2. 사진 칸을 클릭하지 않고 마우스 위치에 따라 사진이 들어갈 수 있도록 수정

    3. 되돌리기 구현

    4. 셀 끼리 사진 교환 가능하게 수정
